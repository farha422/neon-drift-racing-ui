# neon-drift-racing-ui
A futuristic car racing game card UI designed with glassmorphism, featuring a dynamic character, neon effects, and a sleek, interactive interface.
This project showcases a modern car racing game card UI concept, built in Figma using glassmorphism design principles. It includes a custom racing character, neon glow effects, futuristic typography, and a visually appealing layout that can serve as inspiration for game UI or app design projects.
